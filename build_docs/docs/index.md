# Terraform for さくらのクラウド

---

## 概要

`Terraform for さくらのクラウド`とは、
[Terraform](https://terraform.io)から[さくらのクラウド](http://cloud.sakura.ad.jp)を操作するためのTerraform用プラグインです。
